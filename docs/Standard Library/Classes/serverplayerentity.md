# ServerPlayerEntity

Represents a player on the server level. Extends from [PlayerEntity](playerentity.md)

## setGameMode(game_mode: [GameMode](../Enums/gamemodes.md)) -> [Null](null.md) {data-toc-label='setGameMode'}

Changes the players game mode based on `game_mode`.

## dropSelectedItem() -> [Null](null.md) {data-toc-label='dropSelectedItem'}

Drops the players selected item.

## getViewDistance() -> [Integer](integer.md) {data-toc-label='getViewDistance'}

Returns the players view distance.

## setExperienceLevels(experience_levels: [Integer](integer.md)) -> [Null](null.md) {data-toc-label='setExperienceLevels'}

Sets the players experience levels to `experience_levels`.

## setExperiencePoints(experience_points: [Integer](integer.md)) -> [Null](null.md) {data-toc-label='setExperiencePoints'}

Sets the players experience points to `experience_points`.
